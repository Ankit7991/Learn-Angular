Routing Instruction

  1. Add {{steps[1]}} tag to app.component.html
  2. Create app.routing.module.ts file
    • Import all required modules
    • Now insert them in routes variable as follwoing,
    • {{steps[2.3]}}
  3. Import routing module in app.module.ts
  4. Finally use tags {{steps[4]}} inside any html file to initiate routing for that component.
    Mainly put it in app.component.html file to route on all component.
